Home Assistant Mock Device Integration


Project aim: Create a mock device and connect it to Home Assistant. Using Home Assistant MQTT discovery (https://www.home-sisstant.io/integrations/mqtt/#mqtt-discovery, a mock device that sends specified metrics as such temperature and similar data. The mock device should be able to connect and send randomly generated data (the type of data should be by the determined application type, this can can be temperatureless, air pressure, or humidity), HA platform be able to receive this data and print it out. Code should be written in your preffered language: Python, Lua, Javascript (NodeJS) or other
